# Pomodoro App

---

## Quick Start

Preview Link : https://tomatofocus.netlify.app/

## Run Locally

Clone the project

```bash
  git clone https://github.com/prathmeshjagtap/pomodoro-app.git
```

Go to the project directory

```bash
  cd pomodoro-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Features Available.

- Home Page with Info about the App and features
- Tasks Page and Feature
  - Add Task
  - Edit Task
  - Delete Task
  - Mark as Complete
- Pomodoro Feature
  - Add specific Timer
  - use Default Timer
  - Timer according to specific Task
  - Edit Timer
  - Pause Clock
  - Reset Clock
- Authentication
  - Login
  - Sigup
  - Guest Login
- Fully Resposive
- Page title Sync with Pomdoro Time
- Dark Theme Available

---

## Tech Stack

**Client:** React, React Context API + useReducer , React Router v6 , Tailwind

**Server:** Mockbee.
